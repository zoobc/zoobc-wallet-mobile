Overview

ZooBC wallet is a mobile application that is able to connect with the ZooBC blockchain network, so it can carry out activities such as creating accounts, viewing balances, sending coins, displaying QRcode addresses, checking Node status, etc.

Dashboard

In the application dashboard you can see the total balance of all accounts that you have created in this application.

Setting

On the dashboard there are two buttons, namely the Send button and the Receive button. The send button is useful for sending coins from your account to your other account or to someone else's account, you need a destination coin address. Tomobl accept functions to add your coin address along with QRcode. Your friend can scan qrcode and then send a number of coins to you.
