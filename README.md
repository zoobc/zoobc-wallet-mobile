# Features!

  - Create account
  - Wallet Balance 
  - Send Money
  - Scan Barcode
  - Receive Money
  - Buy coin (redirect to marketplace example https://www.coinbase.com/)

### Installation

- Install nodejs 
- npm install -g ionic
- git clone https://github.com/BlockchainZoo/spinechain-wallet-mobile-prototype.git
- cd spinechain-wallet-mobile-prototype
- npm install
- ionic serve
